# React RESTful API 串接

## 作業說明
本專案為 React 後台練習，主要實作登入驗證流程，並在登入成功後顯示產品列表與產品細節。  
專案使用 API 進行登入驗證，並透過 token 控制後台存取權限。

---

## 使用技術
- React
- Vite
- Axios
- Bootstrap 5
- JavaScript (ES6)

---

## 功能列表
- 管理者登入
- 登入狀態驗證（Token 驗證）
- 取得並顯示產品列表
- 查看單一產品詳細內容
- 登出功能（清除 token 並返回登入頁）
